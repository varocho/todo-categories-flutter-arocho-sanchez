# todo-categories-flutter-arocho-sanchez
Esta es una aplicación móvil de manejo de listas de Todos con categorías. La aplicación está pensada para personas que tienen dificultades para organizar sus tareas y quieren llevar un control claro de sus actividades diarias.

Cómo ejecutar la aplicación: 

  Requisitos: 
  
    i. Tener Flutter instalado en tu máquina. 
    ii. Se recomienda una versión mínima de Flutter 3.0 para compatibilidad con Material3 y null safety. 
  
  Pasos principales: 
  
    i. Abrir la terminal en la carpeta del proyecto. 
    ii. Instalar las dependencias con: flutter pub get 
    iii. Ejecutar la aplicación en un emulador o dispositivo conectado: flutter run
